<h2 id="overview">Overview</h2>

This project focuses on analyzing loan distribution, borrower demographics, default behavior, and financial risk metrics using interactive Power BI dashboards. The solution helps identify high-risk borrower segments and supports data-driven lending decisions.

---

<h2 id="business-problem">Business Problem</h2>

Financial institutions need to monitor borrower risk, analyze default trends, and optimize lending strategies to reduce financial losses. This dashboard helps identify risky borrower categories, monitor loan performance, and support data-driven financial decisions.
